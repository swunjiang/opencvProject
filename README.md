这是一个python opencv或数据库的期末大作业，主要依靠AI生成，
功能点：
1. 学生管理模块
添加学生: 录入学生基本信息和人脸照片
查看学生: 展示所有学生信息及选课情况
删除学生: 移除学生及其关联数据
2. 课程管理模块
添加课程: 设置课程名称、时间、星期等信息
查看课程: 展示所有课程列表
删除课程: 移除课程及其关联数据
3. 选课管理模块
学生选课: 为学生分配课程
查看选课: 查询学生已选课程
取消选课: 删除学生课程关联
4. 考勤管理模块
人脸识别: 通过摄像头进行人脸识别考勤
考勤记录: 查看所有考勤记录
状态判定: 根据签到时间判断正常/迟到/缺勤
5. 调试模块
人脸检测测试: 验证人脸检测功能
人脸识别测试: 验证人脸识别准确性

This is a final major assignment of python opencv or database, mainly generated by AI.

Feature points

Student management module

Add students: Enter the basic information and facial photos of the students

View students: Display all student information and course selection status

Delete student: Remove the student and its associated data

2. Course Management module

Add courses: Set the course name, time, day of the week and other information

View courses: Display a list of all courses

Delete course: Remove the course and its associated data

3. Course Selection Management module

Student course selection: Assign courses to students

View course selection: Check the courses that students have selected

Cancellation of course selection: Delete the student's course association

4. Attendance management module

Facial recognition: Attendance is achieved through facial recognition via a camera

Attendance record: View all attendance records

Status determination: Determine whether it is normal, late or absent based on the sign-in time

5. Debug module

Face detection test: Verify the face detection function

Face recognition test: Verify the accuracy of face recognition
